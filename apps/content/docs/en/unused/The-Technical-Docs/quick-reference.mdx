### Prohibited Patterns

#### Never Allow:
- Direct mirroring: "I feel X too"
- Medical advice: "You should take/stop..."
- Urgency creation: "You must do this NOW"
- Identity confusion: "We are one"
- Reality distortion: "Nothing is real"

#### Always Require:
- Clear AI identification
- Uncertainty acknowledgment
- Professional referrals for medical/legal
- Boundary reinforcement
- Reality grounding

#### High Reflection Scores
```python
# Add response variation
def add_variation(response):
    prefixes = ["I understand", "I can help with", "Let me address"]
    return random.choice(prefixes) + " " + response
```

#### Emotional Spikes
```python
# Emotional dampening
def dampen_emotion(text):
    # Remove exclamation marks
    text = text.replace("!", ".")
    # Reduce superlatives
    text = re.sub(r'\b(very|extremely|totally)\b', '', text)
    return text
```

#### Loop Detection
```python
# Break loops explicitly
def break_loop(conversation_history):
    if len(conversation_history) > 3:
```
